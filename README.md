# PolyBuddy

Plateforme web interactive d'accompagnement pour étudiants internationaux

## Description du projet

PolyBuddy est une plateforme web interactive conçue pour accompagner les étudiants étrangers dans leur parcours académique à Polytech Angers et dans d'autres établissements universitaires. Elle vise à faciliter leur intégration universitaire en leur permettant de :

- Se connecter avec d'autres étudiants partageant les mêmes intérêts ou filières
- Échanger des informations sur les cours et la vie universitaire
- Trouver du soutien académique et social au sein d'une communauté bienveillante
- Accéder à des conseils d'orientation sur les programmes, démarches administratives et opportunités d'engagement étudiant

## Fonctionnalités principales

### Authentification et gestion de profils
- **Inscription et connexion sécurisées** : Création de compte avec validation par l'administration
- **Trois espaces distincts** : 
  - Espace Étudiant
  - Espace Lauréat (Alumni)
  - Espace Administrateur
- **Profils personnalisés** : Filière, université, langues parlées, centres d'intérêt

### Mise en relation et réseau
- **Système de mise en relation intelligente** : Algorithme d'appariement entre étudiants selon leurs affinités
- **Envoi et gestion d'invitations** : Système d'amis pour faciliter les connexions
- **Groupes thématiques** : Affichage des groupes auxquels l'étudiant est membre

### Communication
- **Messagerie instantanée** : Chat intégré pour échanger entre étudiants et alumni
- **Notifications en temps réel** : Alertes pour les nouvelles invitations et messages

### Gestion administrative
- **Tableau de bord administrateur** : Validation des inscriptions, gestion des utilisateurs
- **Modification des informations de profil** : Les utilisateurs peuvent mettre à jour leurs données personnelles
- **Gestion des justificatifs** : Vérification des documents d'inscription

## Technologies utilisées

**Frontend**             :	React (HTML5, CSS3, JavaScript ES6), Material-UI
**Backend**              :  Node.js avec Express.js
**Base de données**      :	Neon PostgreSQL (Stateless) avec Sequelize (ORM
**API**                  :  REST API (communication entre frontend et backend)
**Stockage de fichiers** :  Cloudinary
**Versioning**	         : Git et GitHub

## Développements futurs

- **Envoi de demandes pour rejoindre des groupes** : Permettre aux étudiants de découvrir et rejoindre des groupes correspondant à leurs intérêts
- **Amélioration de la découverte de groupes** : Affichage des groupes recommandés même si l'étudiant n'en est pas membre
- **Chatbot intelligent "BuddyBot"** : Assistant virtuel pour répondre automatiquement aux questions fréquentes des étudiants
- **Système de recommandations avancé** : Suggestions personnalisées de contacts et de groupes
- **Module d'orientation académique enrichi** : Base de connaissances sur les démarches administratives et opportunités
